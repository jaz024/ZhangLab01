Definitions for some HTML-related terms:

DOCTYPE statement:
An "information" to the browser about what document type to expect.
How do I use <ol>:
I use <ol> when listing things that are in a specific order
How do I use <ul>:
I use <ul> when listing things that aren't in a specific order
How do I use <dl>:
I use dl when displaying a term and its definition
Difference between a text editor and word processor:
A text editor uses pure text for coding, and a word editor interprets and reformats the code to make it human-readable
client-server model:
A way computers communicate over the internet to get certain results such as a webpage
FTP:
A protocol that allows computer to transfer files in a network.
It allow users upload, download, rename, and delete files on a server
HTTP:
A protocol used for transfering webpages between the webserver and the client.
When we visit a website, our browser uses HTTP to request pages, which the server responds with HTML and other web contents
TCP/IP:
A set of communication rules that defines teh sending and receiving of data on the internet.
TCP ensures reliable delivery by breaking the data into packets, and IP makes sure that the packets goes to the right destination
